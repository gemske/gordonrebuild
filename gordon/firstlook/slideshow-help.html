STEP 1.) Copy Files
Copy the "slideshow-addon" folder from this zip file into your website folder. Copy the 2 html pages into your folder. Open the "slideshow-help.html".


STEP 2.) Add Code to Webpage Head Section
Copy and paste the following code into the "head" of your existing webpage that you want the slideshow on. Put the code just before the </head> near the top of the page. Make backups of your webpages before you start.




STEP 3.) Add Slideshow Code
View the source code of the slideshow-example.htm. Select and copy the code between the notes: "START SLIDESHOW" and "END SLIDESHOW". Paste the code into your HTML page in the content area where you want the slideshow to appear.


STEP 4.) Test
Open your HTML page and test the slideshow. If it's not working review the above steps and make sure you have copied over all files form the zip file.


STEP 5.) Replace Images
Replace the .jpg images in the "slideshow-addon" folder with your slideshow images.


Options
You can add more slides, change colors and set the AutoPlay speed.
  • Copy and paste the bottom image line (slide-6.jpg) in the HTML page to add another slide.
  • Edit the "var rotate_delay = 3000;" in the "slideshow-addon.js" to change the speed.
  • Edit the "slideshow-addon.css" to change button colors and set the slide image border.
